# 初等变换学习辅助工具

一个基于Web的线代学习辅助工具，帮助用户练习矩阵的初等变换操作。

## 快速开始

### 在线使用
[点击链接](https://awei808.github.io/try-error/) 即可在浏览器中使用本工具。
或者打开[网页压缩版](https://awei808.github.io/awei/)，保留全部功能，但更轻量化。

### 本地运行
1. 前往[单网页版仓库](https://github.com/awei808/awei)下载
2. 下载或克隆项目到本地，使用任意现代浏览器打开 `index.html` 文件，无需安装任何依赖，开箱即用
   - 手机、平板由于浏览器限制，请使用第一种方法进行本地运行，或选择在线使用

## 功能特性

### 核心功能
- **矩阵录入**：支持手动录入矩阵元素
- **快速导入**：通过二维数组格式快速导入矩阵
- **初等变换**：支持三种基本初等行变换操作
- **实时预览**：实时显示变换过程和结果

### 矩阵操作
- **行交换**：交换矩阵的两行
- **行倍乘**：将某一行乘以一个非零常数
- **行倍加**：将某一行乘以常数加到另一行

### 用户界面
- **响应式设计**：适配不同屏幕尺寸
- **直观操作**：点击式操作，无需记忆命令
- **状态提示**：清晰的操作指引和状态显示
- **撤销功能**：支持操作撤销，便于学习调试

## 使用指南

### 1. 录入矩阵
- 点击"录入矩阵"按钮
- 在网格中选择矩阵维度
- 输入矩阵元素值

### 2. 执行初等变换
1. 选择目标行/列
2. 选择变换类型（交换、倍乘、倍加）
3. 输入相关参数
4. 点击"执行初等变换"按钮

### 快速导入
- 点击"导入二维数组"按钮
- 输入标准的二维数组格式，如：`[[1,2,3],[4,5,6]]`
- 系统自动解析并显示矩阵

### 导出矩阵
- 通过"更多"菜单中的"导出矩阵为二维数组"功能
- 将当前矩阵数据复制到剪贴板
- 方便在其他应用中使用
- 导出格式为标准的二维数组格式，如：`[[1,2,3],[4,5,6]]`，可直接在快速导入中使用
